# BC Ferries historical conditions

Store the conditions of bc ferries over time so we can use it for analytics

## Features

- **Historical Conditions**: View ferry capacity trends over time for different routes
- **Should I Reserve?**: Get data-driven recommendations on whether to reserve your ferry based on historical patterns
- **Route-Specific SEO Pages**: Dedicated pages for popular routes to improve discoverability

## SEO-Friendly Routes

The following route-specific pages are available:

- `/should-i-reserve/vancouver-victoria` - Vancouver (Tsawwassen) to Victoria (Swartz Bay)
- `/should-i-reserve/victoria-vancouver` - Victoria (Swartz Bay) to Vancouver (Tsawwassen)

These pages provide the same functionality as the main "Should I reserve?" page but are pre-configured for specific routes and optimized for search engines.
